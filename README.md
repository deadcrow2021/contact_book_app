Installing
----------

Clone repository:

```text
$git clone https://github.com/deadcrow2021/nl_test
```

Move to the folder.
Example:

```text
$cd /home/user/Desktop/nl_test/
```

Install required packages:

```text
$pip3 install -r requirements.txt
```

----

How to run
----------

Print for help:

```text
$python manage.py runserver
```
